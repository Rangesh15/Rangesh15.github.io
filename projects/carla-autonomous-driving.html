Autonomous Vehicle Simulation with CARLA

A Python-based CARLA simulation project for vehicle control and camera-feed acquisition using OpenCV.

CARLA Simulator Python OpenCV NumPy Camera Sensors Vehicle Control

Problem

Testing autonomous driving algorithms in the real world is costly and risky. CARLA provides a controlled simulator for vehicle control, perception, and sensor experimentation.

Role

Developer and report author
Course / research project

My Contribution

  • Connected Python scripts to the CARLA simulator server.
  • Spawned vehicles and sensors programmatically.
  • Implemented vehicle control commands and camera-feed processing.
  • Documented the simulation setup, methodology, and results.

System Architecture

  • CARLA server and simulated world
  • Python client script
  • Vehicle actor and camera sensor
  • OpenCV image processing callback
  • Vehicle control loop

Results

  • Demonstrated vehicle spawning, camera acquisition, and control in CARLA.
  • Produced visual simulation outputs and technical documentation.
  • Established a foundation for future autonomous driving experiments.

Why it matters

This project connects core robotics engineering skills—sensing, modeling, control, simulation, and integration—with the type of systems thinking needed for autonomous and humanoid robots.
